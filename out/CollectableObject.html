Class: CollectableObject

CollectableObject()

new CollectableObject()

Creates a new Coin instance at a random x position on the screen. The x position is determined by the coinCounter variable which is incremented each time a new Coin instance is created. The y position is set to a random value between 100 and 230. The coin starts animating immediately.

Methods

animate()

Animates the coin by blinking it on and off. The coin blinks by switching between its two images in the IMAGES array. The blinking occurs every 750 milliseconds. The animation is stored in the intervalIds array to be cleared later.
